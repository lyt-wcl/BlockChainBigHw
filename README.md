# ZJU-blockchain-course-2022

## 如何运行

1. 在本地启动ganache应用。

2. 在 `./contracts` 中安装需要的依赖，运行如下的命令：
    ```bash
    npm install
    ```
3. 在 `./contracts` 中编译合约，运行如下的命令：
    ```bash
    npx hardhat compile
    ```
4. 若步骤3不行，则remix编译运行Myierc20.sol和Test.sol两个合约。
5. 在 `./frontend` 中启动前端程序，运行如下的命令：
    ```bash
    npm run start
    ```
